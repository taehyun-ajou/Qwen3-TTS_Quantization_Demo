Qwen3-TTS Quantization Demo

Calibration-Free Weight-Only Quantization for CPU-Friendly On-Device LLM-based Codec TTS

Audio Demo Results Code Paper

Abstract

This page presents audio samples and evaluation results for quantized Qwen3-TTS. We apply calibration-free weight-only post-training quantization to the LLM backbone and MTP module, while keeping waveform-proximal modules such as the codec decoder in higher precision.

Our experiments show that 4-bit quantization provides a practical trade-off between memory reduction and speech quality, whereas 3-bit and lower settings lead to noticeable degradation.

Model Overview

We quantize the LLM backbone and MTP module because these modules dominate model size and token prediction cost. The codec decoder is kept in higher precision to avoid waveform-level quality degradation.

Quantization Setup

All samples are generated using Qwen3-TTS-12Hz-1.7B-Base. Weight-only quantization is applied to linear layers in the LLM backbone and MTP module. The encoder, speaker encoder, codec decoder, and LM head are kept in higher precision.

Limitations

This demo focuses on calibration-free weight-only post-training quantization. Results may differ when calibration-aware quantization, quantization-aware training, or fine-tuning is applied. CPU inference speed can also vary depending on kernel support and dequantization overhead.
